# 情緒覺察與調節工具

由劉怡君臨床心理師設計開發的情緒覺察與調節工具。

## 功能特色

- 7步驟系統化情緒管理練習
- 身體掃描與認知模式識別
- 情緒命名與調節策略推薦
- 歷史記錄追蹤與視覺化分析

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build
```

## 部署到 Vercel

1. 註冊 [Vercel](https://vercel.com) 帳號
2. 安裝 Vercel CLI: `npm i -g vercel`
3. 在專案目錄執行: `vercel`
4. 跟隨指示完成部署

或直接透過 Vercel 網站上傳專案資料夾。
